# roboteq_hw_interface
new simplified cpp version of hw_interface, and is used to control roboteq controllers with encoder feedback

# Using on YOUR ROBOT
 
 1. COPY the contents of the config/template.yaml and PASTE AT THE END of drivebase_missions/config/drivebase.yaml  
 2. adujust values in drivebase_missions/config/drivebase.yaml  
 3. add roboteq.launch to the hardware launch file in drivebase_missions
